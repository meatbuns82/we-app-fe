# 🍽️ 一起点菜 — 微信小程序

情侣/家庭群组点菜小程序。浏览菜单、点菜、收藏、查看今日订单，支持炒菜和火锅两种模式。

## 技术栈

- **框架**：原生微信小程序（.wxml / .wxss / .js / .json）
- **UI**：WeUI + 自定义 Warm Calm 设计系统
- **后端**：Java Spring Boot REST API

## 页面

| 页面 | 功能 |
|------|------|
| 首页 | 登录、群组切换、功能导航、随机情诗 |
| 菜单 | 菜品浏览/搜索/筛选/点单 |
| 火锅 | 火锅食材区点单 |
| 最爱 | 收藏菜品 |
| 菜品详情 | 烹饪步骤 |
| 购物车 | 订单管理（炒菜/火锅分类） |
| 今日 | 当前群组今日点单总览 |

## 快速开始

1. 用 **微信开发者工具** 打开项目根目录
2. AppID: `wxa39381ef8ce5df39`
3. 确保后端服务已启动（默认 `http://192.168.1.9:9093`）
4. 点击编译预览
